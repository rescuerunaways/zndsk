# zndsk

install dependencies: 
pip install -r requirements.txt

run tests:
python -m unittest url_test

run a server:
./zndsk.py 


Usage:


  zndsk.py show [--page=<number>]


  zndsk.py show <ticket>


  zndsk.py (-h | --help)
  
Example usage:


  ./zndsk.py show


  ./zndsk.py show 2


  ./zndsk.py show --page=2
